Requirements
   Python version 3
   Numpy for Python 3

Instructions:
   To run this program, ensure that the data csv file is in the same folder
   as RuleMining.py. It can be run from the command line with "python3 
   RuleMining.py". It will prompt for a minimum support level before outputting
   the results.
