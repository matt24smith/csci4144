INSTRUCTIONS
   To run this program, execute it from the command line using the Python 3
   interpreter. The sample data set must be placed within the same folder
   as the python script for successful completion.
   The program will save the sorted list to the same folder, and then prompt 
   the user to select an OLAP operation. The resulting aggregation will then
   be printed to STDOUT

   Example execution string:
      python3 ETL-OLAP.py

DEPENDENCIES
   python version: 3.7.2
   numpy version: 1.16.1
