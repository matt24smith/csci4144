DEPENDENCIES
   Python 3
   Pandas
   Matplotlib

INSTRUCTIONS
   To run the program, change directory to the directory containing
   implementation.py and Online_Retail.xlsx. The program can be
   executed with "python3 implementation.py".
   The program will read the transactional records from the excel 
   file, and prompt the user for an alpha value and a stock code.
   The program will output a .png file depicting results in the 
   same folder.
